Ejemplos de etiquetas de agrupacion

Seccion 1

Las etiquetas para estructurar o etiquetas de secciones, es un tipo de etiquetas que permite la agrupación de todos los elementos de un texto, como lo son el título, la introducción, el lead, el contenido, el desarrollo, las conclusiones, entre otros.

Seccion 2

La etiqueta HTML main es una etiqueta que nos permite agrupar todo el contenido principal de una página. En principio, un documento HTML sólo debería tener una etiqueta main . En este caso, sería correcto definir múltiples etiquetas main en un mismo documento.

Seccion 3

La etiqueta hgroup se utiliza para rodear un encabezado y uno o más elementos p . El encabezado dentro del elemento hgroup puede ser cualquiera de los encabezados h1 a h6. Nota: El elemento hgroup no se representa como nada especial en un navegador. Sin embargo, puede usar CSS para darle estilo al elemento hgroup y a su contenido.

Seccion 4

Puedes usar div para agrupar elementos (como texto, imágenes, formularios, etc.) y luego aplicarles estilos CSS, como posicionamiento o espaciado, a todos ellos a la vez. En este ejemplo, la etiqueta div crea un contenedor alrededor de los elementos de encabezado, párrafo e imagen para que se les pueda aplicar estilos juntos.
